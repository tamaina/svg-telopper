# 使い方 (省略版)
サーバーの起動には[**node.js**](https://nodejs.org/ja/)の事前のインストールが必要です(v11で動作確認済み)。

また、[OBS WebSocket](https://github.com/Palakis/obs-websocket/releases/tag/4.4.0)をインストールすることで、OBSとのちょっとした連携を行うことができます。

## はじめる
```
pushd path/to/this/dir
npm start
```

## ビルド
```
npm run build
```
